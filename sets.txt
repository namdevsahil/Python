Ans = my_set = {1,2,3,4,5}

Que 3) How do you add elements to a set in python?
Ans = my_set = {1,2,3}
my_set.add(4)

Que 4) How do you remove elements from a set in python?
Ans = my_set = {1, 2, 3, 4, 5}
my_set.remove(3)
print("After removing 3:", my_set)
